# Extension Icons

Create PNG icons in these sizes:
- `icon16.png` - 16x16 (toolbar)
- `icon48.png` - 48x48 (extensions page)
- `icon128.png` - 128x128 (Chrome Web Store)

**Design suggestion**:
- Purple gradient background (#8b5cf6 to #7c3aed)
- White game controller or castle icon
- Rounded corners

You can use tools like Figma, Canva, or an AI image generator to create these.

For now, the extension will work without icons (Chrome shows a default puzzle piece).
