Quick Actions
Quick Phrases (Ctrl+Alt+#)

MyVMK Commands

Type commands in chat. The leading ! or :: is required.

General Commands
!me (text)
Sends a message in the room with your character name.
!s / !sig / !signature
Updates your signature with any text placed after the command.
!notrades / !toggletrades / !toggledtrading
Stops you from getting trade requests. Type again to re-enable.
!nofriends
Stops you from getting friend requests. Type again to re-enable.
!invisiblemode
Enables invisible mode so you appear offline. Type again or relog to disable.
!lucky
Sends you to a random guest room.
!sniff (user)
Sniffs a user and randomly generates a scent.
!roll #
Rolls a random number between 1 and #.
::screenshot
Downloads a picture of the room (hides chat bubbles).
::screenshot av
Downloads a picture of the room (hides avatars).
::screenshot chat
Downloads a picture of the room (includes chat bubbles).
::presets
Opens a popup interface to view, delete, and edit clothing presets.
::discordtrade
Opens a searchable interface for trades and Discord's #trading-post.
::up
Clears your game window of chats.
::nolanyards
Prevents opening a user's pin lanyard. Type again to re-enable.
::avatars
Lists all users in the same room.
::cratewins
Displays all items you won from crates.
Keyboard Shortcuts
Hold Shift + Click a User
Types the user's name into the chat box.
Hold Shift + Press Next in Messages
Deletes all messages.
Hold Shift + Press - or + in Shops
Bulk adds/subtracts 10 of an item.
Hold Shift + Highlight in Inventory
Allows you to sell the entire stack of a sellable item.
Right Click on Item While Trading
Opens a dropdown for bulk adding items.
Guest Room Commands
!spin
Spins all generators in the current room.
!resetstates rng
Resets only the random number generators to zero.
!resetstates
Resets all items in the room to default state (Generators, Flaming Ransacked Windows, Lanterns, Candles, etc.).
!kick (user)
Kicks a specific user from the guest room.
!q / !queue
Shows users, queue size, and room capacity.
!preload
Toggles loading of all items in a room before entering. Type again to disable.
!cage (username)
Only available in Oogie's Lair guestroom; sends a user to the cage.
Furniture Shortcuts
Hold Shift While Placing Furniture
Places multiples of an item consecutively if you have several in inventory.
Hold Shift + Click a Furniture Item
Puts the item back into your inventory.
Hold Control + Click/Scroll a Furniture Item
Rotates the item before placing it.
