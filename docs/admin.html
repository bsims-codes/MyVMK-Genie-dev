First-time setup instructions
  1. Go to jsonbin.io and create a free account
  2. Click "Create a Bin" and paste: {"version":1,"genieEvents":[],"communityEvents":[]}
  3. Copy the Bin ID from the URL (the part after /b/)
  4. Go to API Keys section and copy your Master Key
  5. Paste both values above and click Connect
