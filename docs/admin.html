Setup Required

Enter your JSONBin.io credentials to enable live event scheduling.

Get these from jsonbin.io (free account)

First-time setup:
  1. Go to jsonbin.io and create a free account
  2. Click "Create a Bin" and paste this: {"version":1,"genieEvents":[],"communityEvents":[]}
  3. Copy the Bin ID from the URL (after /b/)
  4. Go to API Keys, copy your Master Key
  5. Paste both above and click Connect
